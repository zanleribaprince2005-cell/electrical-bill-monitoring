# Electrical Load Monitoring and Billing Simulator

## Course
EEE 227 (PT)

## Programme
HND Electrical Engineering (Level 200)

## Description
This is a C++ console-based application that simulates electrical load monitoring and electricity billing for a small facility.

## Features
- Register appliances
- View appliances
- Search appliance
- Calculate energy consumption
- Calculate electricity billing
- File storage using fstream

## Energy Formula
Energy (kWh) = (Power × Hours) / 1000

## Billing Formula
Total Cost = Total Energy × Tariff

## Files
- main.cpp
- appliances.txt
- billing_summary.txt
